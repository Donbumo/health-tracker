# Primeros pasos

Tras iniciar sesión visita `/getting-started`. La lista se calcula desde datos reales y no bloquea el resto de la app.

1. Configura nombre visible opcional, zona horaria IANA y unidad de carga en `/account/preferences`.
2. Crea una rutina guiada o impórtala desde `/training-plans`.
3. Comprueba la versión activa. Las sesiones históricas siempre conservan su versión original.
4. Planifica un día en `/planned-workouts/new`.
5. Inicia la captura desde Hoy o la agenda.
6. Guarda una sesión y comprueba el historial.
7. Crea un backup ZIP en el centro de datos.
8. Opcionalmente registra un dispositivo API compatible.

El bloque de onboarding deja de ocupar el dashboard cuando detecta datos existentes. Al completar los pasos esenciales puede ocultarse desde la propia guía.
